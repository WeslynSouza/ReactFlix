## 💻 Projeto

Recriar a plataforma da netflix com videos do youtube.

## 🚀 Tecnologias

Este projeto foi desenvolvido com as seguintes tecnologias:

- [Node.js](https://nodejs.org/en/)
- [React](https://reactjs.org)

## 🔖 Layout

Para acessar o layout utilize a ferramenta [figma]

## 📌 Como executar

```sh
git clone https://github.com/WeslynSouza/ReactFlix
npm install
npm start
```
